# Placeholder Images

This directory is for product, industry, and partner logo images.

## Recommended Image Sizes

### Product Images
- **Size**: 800x600px or 4:3 aspect ratio
- **Format**: JPG or PNG
- **File naming**: 
  - `nylon.jpg`
  - `abs.jpg`
  - `pc.jpg`
  - `pmma.jpg`
  - `tpe.jpg`
  - `pom.jpg`
  - `pbt.jpg`
  - `pp.jpg`
  - `masterbatch.jpg`

### Industry Images
- **Size**: 1200x800px or 3:2 aspect ratio
- **Format**: JPG or PNG
- **File naming**:
  - `automotive.jpg`
  - `electrical.jpg`
  - `consumer.jpg`
  - `agricultural.jpg`
  - `medical.jpg`

### Partner/Brand Logos
- **Size**: 300x100px or maintain aspect ratio
- **Format**: PNG with transparent background preferred
- **File naming**:
  - `envalior-logo.png`
  - `lg-logo.png`
  - `samsung-logo.png`
  - `basf-logo.png`
  - etc.

### Certification Images
- **Size**: 800x600px
- **Format**: JPG or PNG
- **File naming**:
  - `gst-cert.png`
  - `envalior-cert.png`

## Note

Currently, the website uses gradient backgrounds and icons as placeholders. When you add actual images, simply place them in this directory with the filenames mentioned above, and they will automatically appear on the website.
